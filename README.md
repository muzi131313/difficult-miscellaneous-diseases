## 📚 项目简介

本仓库是一个开放式的医学知识库，旨在收集和整理常见疾病、疑难杂症的科普知识。内容涵盖病因、症状、诊断、治疗、预防等方面，帮助大众了解疾病知识，提高健康意识。

## 🔥 紧急提醒

如果您或他人出现以下紧急情况，请立即就医或拨打急救电话（120）：

- 💔 剧烈胸痛（可能心肌梗死）
- 🧠 突发意识障碍、肢体无力（可能中风）
- 🤯 突然剧烈头痛
- 😰 严重呼吸困难
- 🩸 大出血
- 🌡️ 高热伴意识模糊

---

## 🛡️ 健康提示

1. **定期体检**：每年至少一次全面体检
2. **健康生活**：均衡饮食、规律运动、充足睡眠
3. **保持心态**：心理健康同样重要
4. **及时就医**：如有不适，及时咨询医生
