## 认识冠心病

### 定义

冠心病（Coronary Artery Disease, CAD）是冠状动脉粥样硬化使血管腔狭窄或阻塞，或因冠状动脉功能性改变（痉挛）导致心肌缺血、缺氧或坏死而引起的心脏病，亦称缺血性心脏病。

### 流行病学

- 全球约有 **1.1亿** 冠心病患者
- 中国冠心病患者约 **1100万**
- 冠心病是全球死亡的主要原因之一
- 每年约 **700万** 人死于冠心病

### 冠心病分类

| 类型 | 说明 |
|------|------|
| **无症状性心肌缺血** | 无明显症状，但检查有心肌缺血证据 |
| **心绞痛** | 胸骨后压榨性疼痛，可放射至左肩、左臂 |
| **心肌梗死** | 冠状动脉阻塞导致心肌坏死 |
| **缺血性心肌病** | 长期缺血导致心肌纤维化、心脏扩大 |
| **猝死** | 突发心脏骤停，多由于心律失常引起 |

### 心绞痛分类

#### 稳定型心绞痛
- 诱因明确：劳累、情绪激动、饱餐、寒冷
- 持续时间：3-5分钟，很少超过15分钟
- 休息或含服硝酸甘油可缓解
- 发作频率相对稳定

#### 不稳定型心绞痛
- 诱因不明显，休息或夜间也可发作
- 持续时间较长，>15分钟
- 含服硝酸甘油缓解效果不佳
- 发作频率增加，疼痛加重
- **属于急症，需立即就医**

## 症状表现

### 典型心绞痛症状

#### 疼痛部位
- 胸骨后（胸骨正中）
- 可放射至：
  - 左肩、左臂
  - 下颌、颈部
  - 背部
  - 上腹部

#### 疼痛性质
- 压榨性、紧缩感
- 像有大石头压在胸口
- "透不过气"
- 不是针刺样疼痛

#### 诱发因素
- 劳累或剧烈运动
- 情绪激动、紧张
- 饱餐（特别是高脂饮食）
- 寒冷天气
- 吸烟

#### 缓解方式
- 休息：停止活动，静坐或平卧
- 药物：含服硝酸甘油

### 心肌梗死症状

#### 剧烈胸痛
- 程度更重，持续时间更长（>15分钟，可持续数小时）
- 休息或含服硝酸甘油不能缓解
- 伴随大汗、濒死感

#### 全身症状
- 恶心、呕吐
- 呼吸困难
- 头晕、乏力
- 严重时可出现意识丧失

#### 心律失常
- 心悸
- 心跳不规律
- 心跳过快或过慢

### 非典型症状

#### 女性常见非典型症状
- 疲劳、乏力
- 消化不良、胃部不适
- 呼吸困难
- 背痛或下颌痛
- 头晕

#### 老年人常见非典型症状
- 乏力、虚弱
- 精神改变（意识模糊）
- 呼吸困难
- 食欲下降

## 病因与风险因素

### 主要风险因素

| 类型 | 风险因素 |
|------|---------|
| **不可控因素** | 年龄（男性>45岁，女性>55岁）、男性性别、遗传家族史 |
| **可控因素** | 高血压、高血脂、糖尿病、吸烟、肥胖、缺乏运动、不健康饮食、精神压力 |

### 9个可控危险因素（Interheart研究）

这9个因素可以解释 **90%** 的首次心肌梗死：

1. **血脂异常**（升高甘油三酯，降低HDL）
2. **吸烟**
3. **糖尿病**
4. **高血压**
5. **腹型肥胖**（腰围：男性>94cm，女性>80cm）
6. **饮食不合理**（少蔬菜水果，高饱和脂肪）
7. **缺乏运动**
8. **心理社会因素**（抑郁、长期压力）
9. **酒精摄入**

### 详细说明

#### 1. 高血脂
- LDL-C（低密度脂蛋白胆固醇）是主要致动脉粥样硬化因素
- 目标：
  - 高危人群（已有冠心病）：LDL-C < 1.8 mmol/L
  - 一般人群：LDL-C < 3.4 mmol/L

#### 2. 吸烟
- 吸烟者患冠心病的风险是不吸烟者的 2-4 倍
- 被动吸烟也增加风险
- 戒烟1年后，风险降低50%

#### 3. 高血压
- 血压≥140/90 mmHg 增加冠心病风险
- 详见：[高血压知识普及](./hypertension.md)

#### 4. 糖尿病
- 糖尿病患者发生冠心病的风险是非糖尿病患者的 2-4 倍
- 详细内容：[糖尿病知识普及](../endocrine/diabetes-type2.md) [待完善]

#### 5. 肥胖
- BMI ≥ 28 kg/m² 明显增加风险
- 腹型肥胖危害更大

#### 6. 缺乏运动
- 缺乏运动是冠心病的独立危险因素
- 规律运动可降低30%风险

## 诊断方法

### 临床表现评估

#### 心绞痛评估（CCS分级）

| 分级 | 表现 |
|------|------|
| I级 | 日常体力活动不引起心绞痛，剧烈运动或长时间劳力才发作 |
| II级 | 日常体力活动轻度受限，快走、爬楼梯诱发心绞痛 |
| III级 | 日常体力活动明显受限，慢走1-2个街区或爬1层楼梯即发作 |
| IV级 | 任何体力活动都可诱发，甚至休息时也可发作 |

### 辅助检查

#### 心电图（ECG）
**静息心电图：**
- 发作时可能显示ST段改变
- 部分患者静息时正常

**运动负荷试验：**
- 平板运动试验
- 可发现隐匿性心肌缺血
- 危险分层

#### 超声心动图
- 评估心脏结构和功能
- 发现室壁运动异常
- 诊断并发症

#### 冠状动脉CT血管造影（CCTA）
- 无创检查
- 显示冠状动脉狭窄程度
- 适用于中危患者

#### 冠状动脉造影（CAG）
**金标准**
- 有创检查
- 准确显示冠状动脉狭窄
- 可同时进行介入治疗

**适应症：**
- 不稳定型心绞痛
- 急性心肌梗死
- 药物治疗无效的心绞痛
- 需要进行危险分层

#### 实验室检查
| 检查项目 | 目的 |
|---------|------|
| 心肌损伤标志物（肌钙蛋白） | 诊断心肌梗死 |
| 血脂 | 评估危险因素 |
| 血糖 | 筛查糖尿病 |
| 肝肾功能 | 评估整体状况 |

### 心肌梗死诊断标准

**必须满足以下至少2条：**

1. 缺血性胸痛症状
2. 心电图动态改变（ST段抬高或压低、T波倒置、病理性Q波）
3. 心肌损伤标志物（肌钙蛋白）升高（超过正常值99%参考上限）

## 治疗方案

### 治疗目标

1. **缓解症状**：控制心绞痛发作
2. **改善预后**：降低心肌梗死和死亡风险
3. **提高生活质量**

### 生活方式干预

#### 1. 饮食调整

**推荐饮食：**
- 地中海饮食
- 低盐、低脂、低糖
- 多吃蔬菜水果
- 全谷物
- 低脂奶制品
- 鱼类（富含ω-3脂肪酸）

**避免食物：**
- 高饱和脂肪：肥肉、动物内脏、黄油
- 反式脂肪：油炸食品、加工食品
- 高胆固醇：蛋黄、海鲜
- 高盐：腌制品、咸菜

#### 2. 运动

**运动建议：**
- **稳定期患者**：规律运动，每周5次，每次30-60分钟中等强度运动
- **急性期或不稳定期**：需在医生指导下进行

**运动类型：**
- 有氧运动：快走、慢跑、游泳、骑自行车
- 抗阻训练：举重、弹力带（稳定期）

**注意事项：**
- 从小运动量开始
- 避免剧烈运动
- 运动时如有胸痛，立即停止

#### 3. 戒烟

- 立即戒烟是最重要的干预措施
- 戒烟1年，心脏病风险降低50%
- 寻求专业帮助（戒烟门诊、戒烟药物）

#### 4. 控制体重

- 目标BMI：18.5-24 kg/m²
- 腰围：男性 < 90cm，女性 < 85cm

#### 5. 心理调节

- 减轻压力
- 学会放松技巧
- 保持良好心态

### 药物治疗

#### 1. 抗血小板药

| 药物 | 作用 | 用法 | 注意事项 |
|------|------|------|---------|
| **阿司匹林** | 抑制血小板聚集 | 75-100mg，每日1次 | 胃肠道反应，餐后服用 |
| **氯吡格雷** | 抑制血小板聚集 | 75mg，每日1次 | 与阿司匹林联用（双抗） |

#### 2. 降脂药

| 药物 | 作用 | 目标 | 副作用 |
|------|------|------|---------|
| **他汀类**（阿托伐他汀、瑞舒伐他汀） | 降低LDL-C，稳定斑块 | LDL-C < 1.8 mmol/L | 肝功能异常、肌肉酸痛 |

#### 3. β受体阻滞剂

| 药物 | 作用 | 用法 | 注意事项 |
|------|------|------|---------|
| **美托洛尔** | 减慢心率，降低心肌耗氧 | 12.5-200mg，每日2次 | 哮喘患者慎用 |
| **比索洛尔** | 减慢心率，降低心肌耗氧 | 2.5-10mg，每日1次 | 心动过缓慎用 |

#### 4. 硝酸酯类

| 药物 | 作用 | 用法 | 注意事项 |
|------|------|------|---------|
| **硝酸甘油** | 扩张冠状动脉，缓解心绞痛 | 心绞痛发作时舌下含服0.5mg | 低血压、头痛 |
| **单硝酸异山梨酯** | 预防心绞痛发作 | 20-40mg，每日2次 | 耐药性 |

#### 5. ACEI/ARB

| 药物 | 作用 | 用法 | 注意事项 |
|------|------|------|---------|
| **培哚普利** | 改善心室重构 | 4-8mg，每日1次 | 咳嗽、高钾 |
| **缬沙坦** | 改善心室重构 | 80-160mg，每日1次 | 高钾 |

#### 6. 钙通道阻滞剂（CCB）

| 药物 | 作用 | 用法 | 注意事项 |
|------|------|------|---------|
| **氨氯地平** | 扩张冠状动脉 | 5-10mg，每日1次 | 踝部水肿 |
| **地尔硫䓬** | 扩张冠状动脉，减慢心率 | 90-180mg，每日1次 | 心动过缓 |

### 介入治疗

#### 经皮冠状动脉介入治疗（PCI）

**适应症：**
- 稳定型心绞痛：药物治疗无效或严重狭窄
- 不稳定型心绞痛：尽早干预
- 急性心肌梗死：急诊PCI（黄金时间窗：发病12小时内）

**方法：**
- 桡动脉或股动脉穿刺
- 导丝和导管到达冠状动脉
- 球囊扩张狭窄血管
- 植入支架（药物洗脱支架）

**优势：**
- 创伤小
- 恢复快
- 缓解症状效果好

**注意事项：**
- 需要长期抗血小板治疗（至少12个月）
- 支架内血栓（罕见但致命）

### 外科治疗

#### 冠状动脉旁路移植术（CABG）

**适应症：**
- 左主干病变（严重）
- 三支病变
- 多支病变合并糖尿病
- PCI失败或复杂病变

**方法：**
- 开胸手术
- 取患者自身血管（乳内动脉、大隐静脉）
- 在狭窄血管远端建立旁路

**优势：**
- 生存率优于PCI（部分患者）
- 远期通畅率高

**劣势：**
- 创伤大
- 恢复慢
- 并发症风险高

### 急性心肌梗死治疗

#### 急救措施

**立即呼叫急救电话（120）**

**等待救护车期间：**
1. 停止活动，静坐或平卧
2. 保持冷静
3. 如果有硝酸甘油，舌下含服
4. 如果有阿司匹林，嚼服300mg
5. 解开衣领，保持呼吸道通畅
6. 如有意识丧失，立即开始心肺复苏

#### 院内治疗

**再灌注治疗（开通闭塞血管）：**
1. **急诊PCI**（首选）
   - 发病12小时内
   - 90分钟内完成

2. **溶栓治疗**（PCI不可及）
   - 发病12小时内
   - 阿替普酶等
   - 溶栓后尽快转运PCI

**药物治疗：**
- 抗血小板：阿司匹林 + 氯吡格雷/替格瑞洛
- 抗凝：低分子肝素
- 他汀类：阿托伐他汀80mg
- β受体阻滞剂
- ACEI/ARB

## 预防措施

### 一级预防（预防发病）

#### 高危人群筛查

**建议筛查人群：**
- 男性 > 40岁，女性 > 50岁
- 有冠心病家族史
- 有3个以上危险因素

**筛查项目：**
- 血脂
- 血压
- 血糖
- 心电图

#### 危险因素控制

| 危险因素 | 目标 | 控制方法 |
|---------|------|---------|
| 吸烟 | 戒烟 | 立即戒烟，避免二手烟 |
| 血压 | <140/90 mmHg | 详见：[高血压](./hypertension.md) |
| 血脂 | LDL-C < 3.4 mmol/L | 低脂饮食，必要时药物 |
| 血糖 | HbA1c < 7%（糖尿病患者） | 控制饮食，降糖药 |
| 体重 | BMI < 24 kg/m² | 饮食控制，运动 |
| 运动 | 每周150分钟中等强度 | 规律运动 |

### 二级预防（预防复发）

#### ABCDE原则

| 字母 | 含义 | 具体措施 |
|------|------|---------|
| **A** | Antiplatelet (抗血小板) | 长期服用阿司匹林 |
| | ACEI | 有适应症者服用ACEI |
| **B** | β-blocker (β受体阻滞剂) | 长期服用β受体阻滞剂 |
| | Blood pressure (血压) | 控制血压 < 140/90 mmHg |
| **C** | Cholesterol (血脂) | LDL-C < 1.8 mmol/L |
| | Cigarette (吸烟) | 戒烟 |
| **D** | Diet (饮食) | 健康饮食 |
| | Diabetes (糖尿病) | 控制血糖 |
| **E** | Exercise (运动) | 规律运动 |
| | Education (教育) | 健康教育 |

#### 康复治疗

**心脏康复项目：**
- 运动训练
- 风险因素管理
- 心理咨询
- 营养指导

**获益：**
- 降低死亡率20-30%
- 改善生活质量
- 减少再住院

## 需要就医的症状

### 紧急情况（立即拨打120）

🔴 **心肌梗死**：出现以下症状

- 剧烈胸痛，持续>15分钟
- 胸痛放射至左肩、左臂、下颌、背部
- 伴大汗、恶心、呕吐
- 伴呼吸困难、心慌
- 伴濒死感

🔴 **不稳定型心绞痛**：
- 静息时心绞痛
- 心绞痛频率增加、程度加重
- 持续时间延长
- 硝酸甘油效果不佳

### 建议就医情况

🟠 以下情况应尽快就医：

- 新出现的心绞痛
- 心绞痛频率增加、程度加重
- 运动耐量下降（走路距离缩短）
- 出现呼吸困难、水肿
- 药物治疗效果不佳

## 自我管理

### 症状日记

记录内容：
- 心绞痛发作时间、持续时间
- 诱发因素
- 疼痛部位、程度
- 缓解方式、药物剂量
- 日常活动情况

### 病情监测

| 项目 | 频率 | 目标 |
|------|------|------|
| 血压 | 每周1-2次 | <140/90 mmHg |
| 血脂 | 每3-6个月 | LDL-C < 1.8 mmol/L |
| 血糖 | 如有糖尿病，定期 | HbA1c < 7% |
| 体重 | 每周 | BMI 18.5-24 |

### 用药管理

- 按时服药，不要漏服
- 使用药盒、提醒APP
- 了解药物作用和副作用
- 定期复诊，调整药物

### 紧急情况处理

**随身携带硝酸甘油：**
- 心绞痛发作时，立即舌下含服
- 立即停止活动，静坐或平卧
- 如5分钟后不缓解，可再含1片
- 如3次仍不缓解，立即拨打120

## 常见误区

### ❌ 误区1：胸痛忍一忍就过去了
**正确认识**：胸痛可能是心绞痛或心肌梗死的信号，特别是劳累或情绪激动后的胸痛，需要及时就医。

### ❌ 误区2：没有症状就不需要治疗
**正确认识**：部分冠心病患者无症状（无症状性心肌缺血），但仍有风险。需要根据检查结果评估和治疗。

### ❌ 误区3：支架植入后就痊愈了
**正确认识**：支架只是解决了局部狭窄，动脉粥样硬化是全身性疾病，仍需长期服药控制危险因素。

### ❌ 误区4：放支架后就不能运动了
**正确认识**：支架植入后，在医生指导下规律运动，反而有助于康复和预防复发。

### ❌ 误区5：一旦做了支架，就再也不能停药
**正确认识**：确实需要长期服药，但不是所有的药都需要终身服用。某些药物（如抗血小板药）需要长期服用，其他药物可根据情况调整。

## 快速参考

### 心绞痛识别

**典型心绞痛：**
- 胸骨后压榨性疼痛
- 劳累或情绪激动诱发
- 休息或含服硝酸甘油缓解（3-5分钟）

**危险信号：**
- 静息时发作
- 持续时间>15分钟
- 伴大汗、呼吸困难
- 硝酸甘油无效

### 急性心肌梗死处理流程

1. **识别症状**：剧烈胸痛>15分钟
2. **立即呼叫120**
3. **等待期间：**
   - 停止活动，静坐或平卧
   - 含服硝酸甘油（如已确诊）
   - 嚼服阿司匹林300mg
   - 如有意识丧失，心肺复苏
4. **急诊就医**：急诊PCI
