## 🔥 紧急情况

如果出现以下症状，请立即就医或拨打急救电话（120）：

### 心肌梗死症状
- 💔 剧烈胸痛，持续超过15分钟
- 💔 疼痛放射至左肩、左臂、下巴或背部
- 💔 伴有呼吸困难、冷汗、恶心呕吐
- 💔 极度焦虑、濒死感

### 中风症状（FAST原则）
- 🧠 Face：面部歪斜
- 🧠 Arm：手臂无力
- 🧠 Speech：言语不清
- 🧠 Time：立即就医

## 📊 流行病学

- 中国心血管病患者约 2.9 亿
- 心血管病死亡率居首位，高于肿瘤及其他疾病
- 每年约有 350 万人死于心血管疾病

## 🛡️ 预防建议

### 生活方式预防
- 戒烟限酒
- 规律运动（每周至少150分钟中等强度运动）
- 健康饮食（低盐、低脂、高纤维）
- 控制体重
- 保持良好心态

### 定期体检
- 测量血压（至少每年一次）
- 检查血脂（至少每2-3年一次）
- 检查血糖（特别是高危人群）
- 心电图检查（必要时）
