# 高血压知识普及

> 高血压是一种常见的慢性疾病，被称为"无声的杀手"。长期高血压会增加心脏病、脑卒中、肾病等严重并发症的风险。

> ⚠️ **重要提示**：本内容仅供科普参考，不能替代专业医疗建议。请阅读仓库中的[免责声明](../DISCLAIMER.md)了解更多详情。

## 认识高血压

### 定义

高血压是指动脉血压持续升高的一种慢性疾病。根据《中国高血压防治指南（2018年修订版）》：

| 分类 | 收缩压（mmHg） | 舒张压（mmHg） |
|------|--------------|--------------|
| 正常血压 | <120 | <80 |
| 正常高值 | 120-139 | 80-89 |
| 高血压 | ≥140 | ≥90 |
| 1级高血压（轻度） | 140-159 | 90-99 |
| 2级高血压（中度） | 160-179 | 100-109 |
| 3级高血压（重度） | ≥180 | ≥110 |
| 单纯收缩期高血压 | ≥140 | <90 |

> 注：在未使用降压药物的情况下，非同日3次测量诊室血压，收缩压≥140 mmHg和/或舒张压≥90 mmHg。

### 流行病学

- 中国高血压患者约 **2.9 亿**
- 每 5 个成年人中就有 **1 个高血压患者**
- 高血压是心脑血管疾病最重要的危险因素
- 每年约有 **350 万人** 死于与高血压相关的疾病

### 高血压分类

**按病因分类：**

1. **原发性高血压**（占95%）
   - 病因不明确
   - 与遗传、生活方式、环境因素有关

2. **继发性高血压**（占5%）
   - 由其他疾病引起
   - 常见原因：
     - 肾脏疾病（肾动脉狭窄、肾小球肾炎）
     - 内分泌疾病（原发性醛固酮增多症、库欣综合征）
     - 血管疾病（主动脉缩窄）
     - 药物因素（口服避孕药、激素类药物）

## 症状表现

### 主要症状

⚠️ **注意**：大多数高血压患者早期无明显症状，被称为"无声的杀手"。

**常见症状：**
- 头痛（后脑勺或前额，早晨明显）
- 头晕、眩晕
- 心悸、心慌
- 耳鸣
- 失眠、多梦
- 烦躁、易怒

### 紧急症状（高血压危象）

如果出现以下症状，请立即就医：

- 🤯 剧烈头痛
- 👁️ 视力模糊、视野缺损
- 😰 呼吸困难
- 💔 胸痛、胸闷
- 🗣️ 言语不清、肢体无力（可能是中风）
- 💓 心跳过快或不规则
- 💦 大量出汗
- 😵 意识模糊、抽搐

**血压水平：**
- 收缩压 ≥ 180 mmHg 和/或 舒张压 ≥ 120 mmHg

### 严重并发症

长期未控制的高血压会导致：

| 并发症 | 症状 |
|-------|------|
| 脑卒中 | 头痛、头晕、肢体无力、言语不清 |
| 冠心病 | 胸痛、心悸、呼吸困难 |
| 心力衰竭 | 活动后气短、下肢水肿 |
| 肾病 | 水肿、尿少、血尿 |
| 视网膜病变 | 视力下降、视野缺损 |

## 病因与风险因素

### 主要风险因素

| 类型 | 风险因素 |
|------|---------|
| **不可控因素** | 年龄（>55岁）、遗传家族史、性别（男性>女性） |
| **可控因素** | 高盐饮食、肥胖、缺乏运动、吸烟、饮酒、精神压力大 |

### 详细说明

#### 1. 高盐饮食
- 钠盐摄入过多会导致水钠潴留，增加血容量
- 建议每日盐摄入量 **< 5克**（约一啤酒瓶盖）
- 注意：酱油、味精、咸菜等含盐量高

#### 2. 肥胖
- 体重指数（BMI）≥ 24 kg/m² 为超重
- BMI ≥ 28 kg/m² 为肥胖
- 腰围：男性≥90cm、女性≥85cm 为中心性肥胖

#### 3. 缺乏运动
- 久坐不动是高血压的独立危险因素
- 建议每周至少 **150分钟** 中等强度运动

#### 4. 吸烟
- 尼古丁导致血管收缩、心率加快
- 吸烟者患高血压的风险是不吸烟者的 2-3 倍

#### 5. 饮酒
- 过量饮酒会升高血压
- 建议男性每日酒精摄入 **< 25g**，女性 **< 15g**

#### 6. 精神压力
- 长期精神紧张、焦虑、抑郁
- 睡眠不足

#### 7. 遗传因素
- 父母有高血压，子女患病风险增加 **30%-50%**

## 诊断方法

### 血压测量

#### 标准测量方法

**准备工作：**
- 测量前 30 分钟避免吸烟、饮咖啡、剧烈运动
- 排空膀胱，安静休息至少 5 分钟
- 坐位，双脚平放地面，背部靠椅背

**正确测量：**
- 上臂与心脏同高
- 袖带大小合适（气囊至少包裹上臂的 80%）
- 测量 2-3 次，取平均值

#### 诊室血压

在医疗环境下测量的血压，由医护人员使用水银血压计或电子血压计测量。

#### 家庭血压监测

**优势：**
- 更能反映真实血压水平
- 发现"白大衣高血压"
- 监测治疗效果

**标准：**
- 早晚各测量 1 次
- 每次测量 2-3 遍，取平均值
- 连续测量 7 天，去掉第一天，计算后 6 天平均值

#### 24小时动态血压监测

佩戴便携式血压计，24小时内定时测量血压。

**优势：**
- 全面了解血压昼夜变化
- 诊断"隐蔽性高血压"
- 评估降压药效果

### 辅助检查

#### 基础检查
| 检查项目 | 目的 |
|---------|------|
| 血常规 | 了解贫血、感染情况 |
| 尿常规 | 了解肾脏情况 |
| 血生化 | 检查血糖、血脂、肾功能、电解质 |
| 心电图 | 了解心脏情况 |
| 超声心动图 | 评估心脏结构和功能 |

#### 进一步检查
| 检查项目 | 目的 |
|---------|------|
| 颈动脉超声 | 检查动脉粥样硬化 |
| 肾脏超声 | 评估肾脏大小和形态 |
| 肾上腺CT | 排查继发性高血压 |
| 眼底检查 | 评估高血压视网膜病变 |

### 诊断标准

**成人高血压诊断：**

- 诊室血压：≥140/90 mmHg，非同日 3 次测量
- 家庭自测血压：≥135/85 mmHg
- 动态血压监测：24小时平均值 ≥130/80 mmHg

## 治疗方案

### 治疗目标

| 人群 | 目标血压（mmHg） |
|------|----------------|
| 一般高血压患者 | <140/90 |
| 糖尿病或慢性肾病患者 | <130/80 |
| 老年患者（≥65岁） | <150/90 |
| 合并心力衰竭/脑卒中 | <130/80 |

### 非药物治疗（生活方式干预）

#### 1. 饮食调整

**DASH饮食（防治高血压饮食）：**

| 食物类别 | 推荐摄入 | 限制摄入 |
|---------|---------|---------|
| 蔬菜水果 | 每天 4-5 份 | - |
| 全谷物 | 每天 6-8 份 | - |
| 低脂奶制品 | 每天 2-3 份 | - |
| 瘦肉、禽肉、鱼类 | 每天 6 份以内 | - |
| 坚果、豆类 | 每周 4-5 份 | - |
| 甜食 | - | 限制在 5 份/周以内 |
| 脂肪 | - | 占总热量 27% 以下 |
| 钠盐 | - | < 5克/天 |

**具体建议：**
- 减少食盐摄入：使用限盐勺、少放盐
- 多吃新鲜蔬菜水果：富含钾、镁
- 选择低脂奶制品：补充钙质
- 限制饱和脂肪：少吃肥肉、动物内脏
- 增加膳食纤维：全谷物、豆类

#### 2. 运动锻炼

**推荐运动：**
- 有氧运动：快走、慢跑、游泳、骑自行车
- 运动强度：中等（能说话但不能唱歌）
- 运动时间：每周至少 150 分钟，或每周 5 次，每次 30 分钟
- 力量训练：每周 2-3 次

**注意事项：**
- 避免剧烈运动
- 从小运动量开始，逐渐增加
- 有并发症时咨询医生

#### 3. 控制体重

**目标：**
- BMI < 24 kg/m²
- 腰围：男性 < 90cm，女性 < 85cm

**方法：**
- 合理饮食
- 规律运动
- 每周减重 0.5-1kg

#### 4. 戒烟限酒

**戒烟：**
- 立即戒烟，避免二手烟
- 寻求专业帮助（戒烟门诊）

**限酒：**
- 男性：酒精摄入 < 25g/天（约 50ml 白酒）
- 女性：酒精摄入 < 15g/天（约 30ml 白酒）
- 高血压患者最好不饮酒

#### 5. 心理调节

- 减轻精神压力
- 学习放松技巧（深呼吸、冥想、瑜伽）
- 保证充足睡眠（7-8 小时/天）
- 培养兴趣爱好

### 药物治疗

#### 降压药分类

| 药物类别 | 常见药物 | 适用人群 | 主要副作用 |
|---------|---------|---------|-----------|
| 利尿剂 | 氢氯噻嗪、吲达帕胺 | 老年人、心力衰竭 | 低钾、尿酸升高 |
| 钙通道阻滞剂（CCB） | 氨氯地平、硝苯地平 | 老年人、心绞痛 | 踝部水肿、面部潮红、牙龈增生 |
| ACEI（血管紧张素转换酶抑制剂） | 贝那普利、培哚普利 | 糖尿病、心力衰竭 | 干咳（10%-20%）、高钾 |
| ARB（血管紧张素Ⅱ受体拮抗剂） | 缬沙坦、厄贝沙坦 | 糖尿病、不能耐受ACEI | 高钾 |
| β受体阻滞剂 | 美托洛尔、比索洛尔 | 心绞痛、心肌梗死后 | 心动过缓、乏力、支气管痉挛 |
| α受体阻滞剂 | 哌唑嗪、特拉唑嗪 | 前列腺增生 | 体位性低血压 |

#### 联合用药

当单药控制不佳时，采用联合用药：

**推荐组合：**
1. 利尿剂 + CCB
2. 利尿剂 + ACEI/ARB
3. CCB + ACEI/ARB
4. CCB + β受体阻滞剂

#### 用药原则

1. **小剂量开始**：从最小有效剂量开始，逐渐调整
2. **长期服用**：高血压需长期甚至终身服药
3. **规律服药**：每天固定时间服药，不要漏服
4. **定期监测**：监测血压，评估疗效和副作用
5. **个体化治疗**：根据年龄、并发症选择药物

#### 常见误区

| 误区 | 正确认识 |
|------|---------|
| 血压正常就可以停药 | 高血压需长期服药，停药会导致反弹 |
| 降压药越贵越好 | 适合的药才是最好的药 |
| 没症状就不需要服药 | 高血压大多无症状，仍需治疗 |
| 药物有依赖性 | 降压药没有成瘾性，但需要长期服用 |
| 相信偏方能根治高血压 | 目前高血压无法根治，需长期控制 |

## 预防措施

### 一级预防（预防发病）

针对高血压高危人群：

1. **健康饮食**
   - 限制盐摄入 < 5克/天
   - 多吃蔬菜水果
   - 减少高脂食物

2. **控制体重**
   - BMI < 24 kg/m²
   - 腰围达标

3. **规律运动**
   - 每周 150 分钟中等强度运动

4. **戒烟限酒**
   - 戒烟，避免二手烟
   - 限制饮酒

5. **心理平衡**
   - 减轻压力
   - 充足睡眠

6. **定期检查**
   - 每年至少测 1 次血压
   - 高危人群每 3-6 个月测 1 次

### 二级预防（早发现、早治疗）

1. **定期监测血压**
   - 家庭自测血压
   - 定期体检

2. **规范治疗**
   - 按时服药
   - 定期随访

3. **控制并发症**
   - 控制血糖、血脂
   - 戒烟限酒
   - 规律运动

### 三级预防（预防并发症和复发）

1. **严格控制血压**：血压达标

2. **全面控制危险因素**
   - 血糖：HbA1c < 7%
   - 血脂：LDL-C < 1.8 mmol/L（高危人群）
   - 戒烟

3. **定期筛查并发症**
   - 心脏：心电图、超声心动图
   - 脑血管：颈动脉超声
   - 肾脏：尿常规、肾功能
   - 眼底：眼底检查

## 需要就医的症状

### 紧急情况（立即就医）

🔴 **高血压危象**：血压 ≥180/120 mmHg 伴随以下症状：

- 剧烈头痛
- 胸痛、胸闷
- 呼吸困难
- 言语不清、肢体无力
- 视力模糊
- 意识模糊、抽搐

### 建议就医情况

🟠 以下情况应尽快就医：

- 血压持续升高，自行调整药物无效
- 出现新的症状（头晕、心悸、水肿）
- 服用降压药后出现严重副作用
- 准备怀孕或已经怀孕
- 有高血压并发症迹象（蛋白尿、眼底出血等）

## 特殊人群注意事项

### 老年高血压

**特点：**
- 收缩压高、舒张压正常或偏低（脉压大）
- 血压波动大
- 容易发生体位性低血压

**治疗建议：**
- 目标血压：收缩压 < 150 mmHg
- 优先选择 CCB、利尿剂
- 小剂量开始，缓慢调整
- 注意体位性低血压

### 儿童青少年高血压

**常见原因：**
- 继发性高血压（肾动脉狭窄、肾小球肾炎）
- 肥胖
- 家族史

**诊断标准：**
| 年龄 | 收缩压（mmHg） | 舒张压（mmHg） |
|------|--------------|--------------|
| 3-5岁 | ≥110 | ≥70 |
| 6-9岁 | ≥120 | ≥80 |
| 10-13岁 | ≥125 | ≥85 |
| ≥14岁 | ≥140 | ≥90 |

**治疗建议：**
- 优先治疗基础疾病
- 生活方式干预为主
- 药物治疗需谨慎

### 妊娠期高血压

**分类：**
1. 妊娠期高血压：妊娠20周后血压 ≥140/90 mmHg
2. 妊娠期高血压疾病（子痫前期）：血压升高 + 蛋白尿或器官功能损害

**治疗建议：**
- 目标血压：130-140/80-90 mmHg
- 安全药物：拉贝洛尔、甲基多巴、硝苯地平
- 禁用：ACEI、ARB（致畸）

**紧急情况**：
- 血压 ≥160/110 mmHg：立即就医，可能需要住院
- 子痫：立即终止妊娠

### 糖尿病合并高血压

**治疗建议：**
- 目标血压：<130/80 mmHg
- 优先选择 ACEI 或 ARB（保护肾脏）
- 联合用药：ACEI/ARB + CCB 或利尿剂
- 控制血糖：HbA1c < 7%

## 自我管理

### 家庭血压监测

**测量频率：**
- 初诊或调整药物：每日早晚各1次，连续7天
- 血压达标：每周1-2天，每日早晚各1次

**记录内容：**
- 日期、时间
- 血压值（收缩压/舒张压）
- 心率
- 测量前活动情况
- 服药时间

### 生活方式记录

| 项目 | 记录内容 |
|------|---------|
| 饮食 | 盐摄入、饮酒量 |
| 运动 | 类型、时间、强度 |
| 体重 | 每周测量1次 |
| 睡眠 | 睡眠时间、质量 |
| 压力 | 压力水平、应对方式 |

### 用药管理

- 使用分药盒，避免漏服
- 设置服药提醒
- 记录服药时间
- 复诊时向医生汇报副作用

## 常见误区

### ❌ 误区1：没有症状就不需要治疗
**正确认识**：大多数高血压早期无症状，但仍在损害器官。即使没有症状，也需要治疗。

### ❌ 误区2：血压正常就可以停药
**正确认识**：血压正常是药物控制的结果。停药会导致血压反弹，增加风险。需长期甚至终身服药。

### ❌ 误区3：降压药有依赖性，越吃越离不开
**正确认识**：降压药没有成瘾性。高血压本身需要长期控制，而不是药物有依赖性。

### ❌ 误区4：降压药越贵越好
**正确认识**：适合的药才是最好的药。药物选择应根据个人情况，由医生决定。

### ❌ 误区5：相信偏方能根治高血压
**正确认识**：目前高血压无法根治，需要长期控制。偏方可能延误治疗，甚至有害。

### ❌ 误区6：降压药伤肾
**正确认识**：正确使用降压药，特别是ACEI和ARB，反而保护肾脏。不治疗高血压才会导致肾损害。

| 危险分层 | 血压水平（mmHg） | 风险程度 |
|---------|---------------|---------|
| 低危 | 1级高血压 + 无危险因素 | 未来10年心血管事件风险 < 10% |
| 中危 | 1级高血压 + ≥1个危险因素<br>2级高血压 + 无危险因素 | 未来10年心血管事件风险 10%-20% |
| 高危 | 1-2级高血压 + ≥3个危险因素或靶器官损害<br>3级高血压 | 未来10年心血管事件风险 20%-30% |
| 极高危 | 临床并发症或合并糖尿病 | 未来10年心血管事件风险 > 30% |

### 紧急情况处理

**高血压危象（≥180/120 mmHg）伴症状：**

1. 立即拨打急救电话（120）或前往急诊
2. 保持安静，平卧或半卧位
3. 不要自行服用降压药（医生会根据情况处理）
4. 如有意识障碍，保持呼吸道通畅
