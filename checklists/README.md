## 🔍 如何使用检查清单

### 使用步骤

1. **选择合适的清单**：根据您关心的健康问题选择相应的清单
2. **逐项检查**：仔细阅读每一项，根据实际情况勾选
3. **评估结果**：根据清单说明评估您的健康状况
4. **采取行动**：
   - **低风险**：保持健康生活方式，定期体检
   - **中等风险**：改善生活方式，尽快就医咨询
   - **高风险**：立即就医，进行全面检查

### 使用频率建议

| 清单类型 | 频率 | 说明 |
|---------|------|------|
| 风险评估清单 | 每年1次 | 定期评估健康风险 |
| 症状检查清单 | 出现症状时 | 帮助判断是否需要就医 |
| 健康监测清单 | 每季度1次 | 监控健康状况变化 |

---

## ⚠️ 重要提醒

1. **仅供参考**：检查清单仅供参考，不能替代专业医疗诊断
2. **及时就医**：如有严重症状或高危因素，及时就医
3. **定期体检**：建议每年至少一次全面体检
4. **结合实际**：检查结果需结合个人实际情况综合判断
5. **咨询医生**：对检查结果有疑问，咨询医生

## 📊 检查清单说明

### 心血管风险自测清单

**适用人群：**
- 所有成年人
- 有心血管疾病家族史者
- 40岁以上人群

**评估内容：**
- 基础风险因素（年龄、性别、家族史）
- 可控风险因素（血压、血脂、血糖、吸烟、肥胖等）
- 生活方式（运动、饮食、压力）

**结果解读：**
- 低风险：继续保持健康生活方式
- 中等风险：改善生活方式，定期监测
- 高风险：立即就医，进行全面检查

### 脑梗风险自测清单

**适用人群：**
- 50岁以上人群
- 有脑梗家族史者
- 有高血压、糖尿病、心脏病者

**评估内容：**
- 基础风险因素
- 症状评估
- 危险因素计数

**结果解读：**
- 低风险：定期监测
- 中等风险：改善生活方式，咨询医生
- 高风险：立即就医，考虑预防用药

### 消化系统健康检查清单

**适用人群：**
- 所有成年人
- 有消化系统症状者
- 饮食不规律者

**评估内容：**
- 饮食习惯
- 症状频率
- 生活方式

**结果解读：**
- 健康：继续保持
- 需改善：调整饮食和生活习惯
- 需就医：及时就医检查
