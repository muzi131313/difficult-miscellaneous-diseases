## 认识脑梗

### 定义

脑梗死（Cerebral Infarction）又称缺血性脑卒中，是由于脑部血液供应障碍，导致脑组织缺血、缺氧，进而引起局限性脑组织缺血性坏死或软化。

**相关术语：**
- **脑梗塞**：脑梗死的旧称
- **缺血性脑卒中**：脑梗的医学术语
- **脑血栓形成**：脑梗的一种类型

### 流行病学

- 中国脑卒中（含脑梗）患者约 **1300万**
- 每年新发脑卒中约 **250万**
- 脑卒中是导致死亡和残疾的主要原因之一
- **病发年龄**：主要在 **45-90岁** 之间，但近年有年轻化趋势

### 脑梗分类

#### 按病因分类

| 类型 | 比例 | 说明 |
|------|------|------|
| **动脉粥样硬化性脑梗死** | 约40% | 动脉粥样硬化形成血栓 |
| **心源性脑梗死** | 约15-20% | 心脏血栓脱落（如房颤） |
| **小动脉闭塞性脑梗死** | 约25-30% | 腔隙性脑梗死 |
| **其他原因** | 约10-15% | 动脉炎、血液病、夹层等 |
| **原因不明** | 约10-15% | 隐源性脑梗死 |

#### 按发病机制分类

1. **血栓形成**
   - 在动脉粥样硬化斑块基础上形成血栓
   - 常见于颈动脉、大脑中动脉

2. **栓塞**
   - 血栓从其他部位脱落，随血流堵塞脑动脉
   - 常见于心源性栓塞（房颤、心肌梗死）

3. **血流动力学改变**
   - 血压下降、血容量不足导致低灌注
   - 常见于严重脱水、休克

#### 与脑出血的区别

| 特点 | 脑梗（缺血性） | 脑出血（出血性） |
|------|--------------|----------------|
| 发病速度 | 相对较慢（分钟到小时） | 急骤（秒到分钟） |
| 主要症状 | 偏瘫、失语、感觉障碍 | 剧烈头痛、呕吐、昏迷 |
| 早期血压 | 可能正常或轻度升高 | 明显升高 |
| CT早期表现 | 可能正常或低密度影 | 高密度影（出血灶） |
| 治疗原则 | 溶栓、抗栓、扩血管 | 控制血压、手术清除血肿 |

## 症状表现

### 典型症状

#### 突然出现的神经功能缺损

**常见表现：**
- 一侧肢体无力或麻木
- 口角歪斜、流口水
- 言语不清或失语
- 视力模糊或视物重影
- 头晕、平衡障碍
- 意识障碍

#### 不同部位梗死的症状

| 梗死部位 | 主要症状 |
|---------|---------|
| **颈内动脉系统** | 偏瘫、失语、偏身感觉障碍 |
| **椎-基底动脉系统** | 眩晕、复视、吞咽困难、共济失调 |
| **大脑半球** | 失语、偏盲、偏瘫 |
| **脑干** | 交叉性瘫痪、意识障碍 |

### 快速识别（FAST原则）

⚠️ **牢记FAST原则，快速识别脑梗：**

| 字母 | 含义 | 检查方法 |
|------|------|---------|
| **F** - Face | 面部 | 让患者微笑，观察是否一侧面部下垂或歪斜 |
| **A** - Arm | 手臂 | 让患者平举双臂，观察是否一侧手臂无力下垂 |
| **S** - Speech | 言语 | 让患者重复简单语句，观察是否言语不清或困难 |
| **T** - Time | 时间 | 出现以上任何症状，立即拨打120，时间就是大脑 |

### 紧急症状（需立即就医）

🔴 如果出现以下症状，**立即拨打急救电话（120）**：

1. **突然发生的严重症状**
   - 剧烈头痛（尤其是伴有呕吐）
   - 突然意识模糊或昏迷
   - 癫痫发作

2. **神经功能缺损**
   - 一侧肢体完全无力或麻木
   - 说话完全无法理解或表达
   - 一侧视力完全丧失

3. **进行性加重**
   - 症状在数分钟或数小时内迅速加重

### 后遗症表现

脑梗后可能出现的后遗症：

- **运动功能障碍**：偏瘫、肌张力增高
- **感觉障碍**：麻木、疼痛、感觉缺失
- **言语障碍**：失语、构音障碍
- **认知功能障碍**：记忆力下降、注意力不集中
- **吞咽障碍**：饮水呛咳、吞咽困难
- **情绪障碍**：抑郁、焦虑
- **大小便失禁**

## 病因与风险因素

### 主要风险因素

| 类型 | 风险因素 | 相对风险 |
|------|---------|---------|
| **不可控因素** | 年龄（>55岁） | 年龄每增加10岁，风险翻倍 |
| | 男性性别 | 男性风险高于女性 |
| | 遗传/家族史 | 有家族史者风险增加2-3倍 |
| **可控因素** | 高血压 | 最重要危险因素，风险增加4倍 |
| | 糖尿病 | 风险增加2-3倍 |
| | 心房颤动（房颤） | 风险增加5倍 |
| | 高血脂 | 风险增加1.5-2倍 |
| | 吸烟 | 风险增加2-4倍 |
| | 肥胖 | 风险增加1.5-2倍 |
| | 缺乏运动 | 风险增加1.5倍 |
| | 过量饮酒 | 风险增加1.5-2倍 |

### 详细说明

#### 1. 高血压

**危害：**
- 长期高血压导致动脉粥样硬化
- 增加血栓形成风险
- 是脑梗最重要的危险因素

**管理：**
- 详见：[高血压知识普及](../cardiovascular/hypertension.md)

#### 2. 心脏疾病

**房颤：**
- 心房颤动时，心房内易形成血栓
- 血栓脱落可导致脑栓塞
- 风险增加 **5倍**

**其他心脏病：**
- 心肌梗死
- 风湿性心脏病
- 心脏瓣膜病
- 先天性心脏病

#### 3. 糖尿病

**危害：**
- 导致血管内皮损伤
- 加速动脉粥样硬化
- 增加血液黏稠度

**管理：**
- 详见：[糖尿病知识普及](../endocrine/diabetes-type2.md) [待完善]

#### 4. 高血脂

**危害：**
- LDL-C升高导致动脉粥样硬化
- 形成不稳定斑块，易破裂形成血栓

**管理：**
- 控制LDL-C < 2.6 mmol/L（高危人群 < 1.8 mmol/L）

#### 5. 吸烟

**危害：**
- 损伤血管内皮
- 增加血液黏稠度
- 促进血栓形成

**戒烟益处：**
- 戒烟1年后，脑梗风险降低50%
- 戒烟5年后，风险接近不吸烟者

#### 6. 饮食习惯

**危险饮食：**
- 高盐饮食（>6g/天）
- 高脂肪、高胆固醇饮食
- 喜食肥肉、动物内脏
- 缺乏蔬菜水果

**健康饮食：**
- 低盐、低脂
- 多吃蔬菜水果
- 全谷物
- 鱼类（富含ω-3）

#### 7. 缺乏运动

**危害：**
- 导致肥胖、高血压、高血脂
- 血管弹性下降

**建议：**
- 每周至少150分钟中等强度运动
- 快走、慢跑、游泳等

#### 8. 精神压力

**危害：**
- 导致血压升高
- 增加血液黏稠度
- 诱发血管痉挛

**建议：**
- 学会放松技巧
- 保证充足睡眠
- 培养兴趣爱好

## 诊断方法

### 临床评估

#### 病史询问
- 症状发生时间（决定治疗窗口）
- 症状特点和进展
- 既往病史（高血压、糖尿病、心脏病等）
- 用药史

#### 体格检查
- 神经系统检查（运动、感觉、反射、颅神经）
- 心脏检查（心脏杂音、心律）
- 血管检查（颈动脉杂音）

### 辅助检查

#### 影像学检查

**1. 头颅CT**
- **首选检查**
- 早期（6小时内）可能正常
- 24小时后可见低密度梗死灶
- 主要用于排除脑出血

**2. 头颅MRI**
- **更敏感**
- 早期（发病1-2小时）即可显示梗死灶
- DWI序列是最敏感的方法
- 适用于发病时间不明或CT阴性的患者

**3. 血管影像检查**
- **CTA（CT血管造影）**：显示血管狭窄或闭塞
- **MRA（磁共振血管造影）**：显示血管情况
- **DSA（数字减影血管造影）**：金标准，可同时介入治疗

#### 实验室检查

| 检查项目 | 目的 |
|---------|------|
| 血常规 | 感染、贫血 |
| 血糖、糖化血红蛋白 | 评估糖尿病控制 |
| 血脂 | 评估危险因素 |
| 凝血功能 | 评估凝血状态 |
| 心肌损伤标志物 | 排除心肌梗死 |
| 同型半胱氨酸 | 危险因素 |

#### 心脏检查

| 检查项目 | 目的 |
|---------|------|
| 心电图 | 评估心律失常 |
| 心脏超声 | 检查心脏结构、功能 |
- 24小时动态心电图 | 检测房颤 |

### 诊断标准

**脑梗死诊断标准（中国急性缺血性脑卒中诊治指南2018）：**

1. **急性起病**
2. **局灶性神经功能缺损**（一侧面部或肢体无力、麻木、言语障碍等）
3. **排除非血管性病因**
4. **影像学证据**（CT/MRI显示梗死灶）或症状持续 > 24小时

**排除脑出血：**
- CT早期即可排除脑出血

## 治疗方案

### 急性期治疗（发病4.5小时内）

#### 1. 静脉溶栓

**适应症：**
- 年龄 ≥ 18岁
- 诊断明确为缺血性脑卒中
- 发病时间 ≤ 4.5小时
- NIHSS评分 ≥ 4分
- 患者或家属知情同意

**禁忌症：**
- 颅内出血史
- 近期重大手术或外伤
- 活动性出血
- 血小板 < 100×10^9/L
- 凝血功能异常

**常用药物：**
- **阿替普酶（rt-PA）**：0.9 mg/kg，最大90mg
- 尿激酶：100-150万IU

**效果：**
- 约30%患者获得良好预后
- 风险：颅内出血（约6%）

#### 2. 血管内治疗

**适应症：**
- 大血管闭塞（颈动脉、大脑中动脉）
- 发病时间 ≤ 6小时（前循环）
- 发病时间 ≤ 24小时（后循环，符合影像学标准）

**方法：**
- 机械取栓
- 动脉溶栓

**优势：**
- 再通率高（>80%）
- 适用于大血管闭塞

### 急性期治疗（发病4.5小时后）

#### 一般治疗

**1. 基础治疗**
- 保持呼吸道通畅
- 吸氧（血氧<95%）
- 控制血压（>180/110mmHg时降压）
- 控制血糖（7.8-10.0 mmol/L）
- 维持水电解质平衡

**2. 防治并发症**
- 颅内压增高
- 感染（肺炎、尿路感染）
- 深静脉血栓
- 压疮

#### 抗血小板治疗

**药物选择：**
- **阿司匹林**：150-300mg负荷量，此后100mg/天
- **氯吡格雷**：75mg/天（不耐受阿司匹林时）

**注意：**
- 溶栓后24小时内不使用抗血小板药
- 不推荐双重抗血小板（除非特殊病因）

#### 抗凝治疗

**适应症：**
- 心源性脑梗死（房颤）
- 其他高凝状态

**药物选择：**
- **华法林**：目标INR 2.0-3.0
- **达比加群**：110-150mg，每日2次
- **利伐沙班**：15-20mg，每日1次

**注意：**
- 溶栓后24小时内不使用
- 需要定期监测INR（华法林）

#### 他汀类药物

**作用：**
- 降低LDL-C
- 稳定斑块
- 抗炎

**推荐：**
- 阿托伐他汀20-80mg/天
- 瑞舒伐他汀10-40mg/天
- 目标：LDL-C < 1.8 mmol/L

#### 降脂稳斑治疗

**药物：**
- **他汀类**：
  - 阿托伐他汀
  - 瑞舒伐他汀
  - 辛伐他汀
- **依折麦布**：与他汀联用

#### 脑保护治疗

**药物：**
- **依达拉奉**：清除自由基
- **胞磷胆碱**：促进脑代谢

#### 改善脑循环

**药物：**
- 银杏叶制剂
- 尼莫地平
- 丁苯酞

### 二级预防（预防复发）

#### 血压控制

**目标血压：**
- 一般患者：< 140/90 mmHg
- 合并糖尿病或肾病：< 130/80 mmHg

**药物选择：**
- 详见：[高血压知识普及](../cardiovascular/hypertension.md)

#### 血脂管理

**目标：**
- LDL-C < 1.8 mmol/L（极高危）
- LDL-C < 2.6 mmol/L（高危）

**药物：**
- 他汀类药物长期服用

#### 血糖控制

**目标：**
- HbA1c < 7%
- 空腹血糖 4.4-7.0 mmol/L

#### 抗血小板治疗

**长期服用：**
- 阿司匹林 100mg/天
- 氯吡格雷 75mg/天（不耐受阿司匹林）

#### 生活方式干预

- 戒烟
- 限酒
- 规律运动
- 健康饮食
- 控制体重
- 心理平衡

### 康复治疗

#### 康复时机

- 生命体征稳定后即可开始
- 早期（发病后1-3个月）是康复黄金期

#### 康疗内容

**1. 运动功能康复**
- 肢体功能训练
- 关节活动度训练
- 平衡和协调训练

**2. 言语功能康复**
- 发音训练
- 语言理解训练
- 构音障碍训练

**3. 吞咽功能康复**
- 吞咽训练
- 饮食调整（如需要）

**4. 认知功能康复**
- 记忆力训练
- 注意力训练
- 执行功能训练

**5. 心理康复**
- 心理咨询
- 支持性治疗

## 预防措施

### 一级预防（预防发病）

#### 针对高危人群

**筛查对象：**
- 年龄 > 55岁
- 有高血压、糖尿病、心脏病
- 有脑卒中家族史
- 吸烟、肥胖、缺乏运动

**筛查项目：**
- 血压监测
- 血脂检查
- 血糖检查
- 心电图、心脏超声
- 颈动脉超声

#### 危险因素控制

| 危险因素 | 目标 | 控制方法 |
|---------|------|---------|
| 高血压 | <140/90 mmHg | 降压药、生活方式干预 |
| 高血脂 | LDL-C < 2.6 mmol/L | 他汀类、饮食控制 |
| 糖尿病 | HbA1c < 7% | 降糖药、饮食控制 |
| 吸烟 | 戒烟 | 立即戒烟，避免二手烟 |
| 肥胖 | BMI < 24 kg/m² | 饮食控制、运动 |
| 房颤 | 规律治疗 | 抗凝药、控制心室率 |

### 二级预防（预防复发）

#### ABCDE原则

| 字母 | 含义 | 具体措施 |
|------|------|---------|
| **A** | Antiplatelet (抗血小板) | 阿司匹林或氯吡格雷 |
| | Anticoagulation (抗凝) | 房颤患者使用华法林或新型口服抗凝药 |
| **B** | Blood pressure (血压) | 控制血压 < 140/90 mmHg |
| **C** | Cholesterol (血脂) | LDL-C < 1.8 mmol/L |
| | Cigarette (吸烟) | 戒烟 |
| **D** | Diet (饮食) | 健康饮食 |
| | Diabetes (糖尿病) | 控制血糖 |
| **E** | Exercise (运动) | 规律运动 |
| | Education (教育) | 健康教育 |

#### 特殊情况处理

**颈动脉狭窄：**
- 狭窄>70%：考虑颈动脉内膜切除术或支架植入
- 狭窄<50%：药物治疗为主

**卵圆孔未闭（PFO）：**
- 反复不明原因脑梗死：考虑封堵术

## 需要就医的症状

### 紧急情况（立即拨打120）

🔴 **出现FAST症状：**
- **F**ace：面部歪斜
- **A**rm：手臂无力
- **S**peech：言语不清
- **T**ime：立即就医

🔴 **其他紧急症状：**
- 剧烈头痛
- 意识模糊或昏迷
- 癫痫发作
- 进行性加重的神经功能缺损

### 建议就医情况

🟠 以下情况应尽快就医：

- 出现新的神经功能缺损症状
- 原有症状加重
- 二级以上医院国家规定的都设立康复科，可以寻求康复治疗

## 特殊人群注意事项

### 老年人

**特点：**
- 基础疾病多（高血压、糖尿病等）
- 血管脆性大
- 康复能力差

**注意事项：**
- 积极控制危险因素
- 预防跌倒
- 早期康复

### 中青年

**特点：**
- 往往忽视危险因素
- 压力大、生活不规律

**注意事项：**
- 定期体检
- 健康生活方式
- 关注高血压、糖尿病

### 孕妇

**特殊考虑：**
- 妊娠期高血压综合征风险
- 用药需谨慎（某些药物禁忌）

### 儿童和青少年

**罕见原因：**
- 先天性心脏病
- 血液病
- 血管畸形

## 中医治疗

### 中医认识

**病名：**
- 中风
- 卒中

**病因：**
- 风、火、痰、虚、瘀

**病机：**
- 气血逆乱，上犯于脑

### 中医辨证分型

| 分型 | 症状 | 治法 |
|------|------|------|
| **风痰瘀血** | 半身不遂，言语不利 | 活血化瘀，祛风通络 |
| **气虚血瘀** | 神疲乏力，面色苍白 | 益气活血 |
| **肝肾阴虚** | 头晕耳鸣，腰膝酸软 | 滋补肝肾 |
| **痰热腑实** | 便秘，舌苔黄腻 | 化痰通腑 |

### 常用中药

#### 1. 活血化瘀药

**三七粉**
- **功效**：活血化瘀、消肿止痛
- **用法**：3-5g/天，温水冲服
- **注意事项**：孕妇忌用，出血性疾病慎用
- **参考**：[三七粉（百度百科）](https://baike.baidu.com/item/%E4%B8%89%E4%B8%83%E7%B2%89/6038480)

**丹参**
- **功效**：活血化瘀、凉血消痈
- **用法**：9-15g/天，水煎服
- **制剂**：丹参滴丸、复方丹参片

**红花**
- **功效**：活血通经、祛瘀止痛
- **用法**：3-9g/天

#### 2. 祛风通络药

**天麻素**
- **功效**：平肝息风、止痉
- **用法**：50-100mg/次，每日3次
- **作用**：
  - 原料产地为云南
  - 具有较好的镇静和安眠作用
  - 对神经衰弱、失眠、头痛症状有缓解作用
  - 中药天麻可治疗眩晕、肢体麻木、惊痫抽搐
- **参考**：[天麻素（百度百科）](https://baike.baidu.com/item/%E5%A4%A9%E9%BA%BB%E7%B4%A0/6165166)

**川芎**
- **功效**：活血行气、祛风止痛
- **用法**：3-9g/天

#### 3. 益气养血药

**黄芪**
- **功效**：补气升阳、益卫固表
- **用法**：9-30g/天

**当归**
- **功效**：补血活血、调经止痛
- **用法**：6-12g/天

### 中医食疗

#### 常用食疗方

**1. 山楂三七丹参粉**
- **原料**：生山楂一斤、生三七一斤、生丹参一斤
- **方法**：洗净、晾干，研磨为粉末
- **用法**：温水冲服，每天早中晚各一杯
- **功效**：降血脂、防三高
- **参考**：[老中医降血脂、防三高的秘方！](http://www.360doc.com/content/17/0414/18/35854156_645617352.shtml)

**2. 决明子茶**
- **原料**：决明子10-15g
- **方法**：开水冲泡代茶饮
- **功效**：清肝明目、润肠通便、调节血压血脂
- **注意事项**：孕妇忌用，低血压者慎用
- **参考**：[决明子（医学百科）](http://www.a-hospital.com/w/%E5%86%B3%E6%98%8E%E5%AD%90)

**3. 姬松茸芦笋汤**
- **原料**：姬松茸、芦笋
- **功效**：养血管、降血脂
- **注意**：姬松茸、芦笋的益生菌发酵液也有协助脑梗治疗作用

#### 饮食建议

**推荐食物：**
- 蔬菜：芹菜、菠菜、洋葱、大蒜
- 水果：苹果、香蕉、猕猴桃
- 全谷物：燕麦、糙米、全麦面包
- 低脂奶制品
- 鱼类（富含ω-3脂肪酸）

**限制食物：**
- 高盐：腌制品、咸菜
- 高脂：肥肉、动物内脏
- 高糖：甜食、含糖饮料

**饮食原则：**
- 蔬菜、水果、低脂奶制品及总脂肪和饱和脂肪含量较低
- 低盐、低脂、低糖
- 少量多餐
- 细嚼慢咽

## 康复指导

### 早期康复（发病后1-4周）

#### 目标
- 预防并发症
- 促进功能恢复
- 提高生活质量

#### 内容

**1. 良肢位摆放**
- 定时翻身（每2小时）
- 防止关节挛缩
- 预防压疮

**2. 被动运动**
- 关节活动度训练
- 预防关节僵硬
- 防止深静脉血栓

**3. 主动运动**
- 早期进行床边活动
- 逐渐增加活动量

### 中期康复（发病后1-6个月）

#### 目标
- 提高运动功能
- 改善日常生活能力
- 促进言语功能恢复

#### 内容

**1. 运动功能训练**
- 平衡训练
- 步态训练
- 上肢精细动作训练

**2. 日常生活能力训练**
- 进食、穿衣、如厕
- 洗澡、转移
- 家务活动

**3. 言语功能训练**
- 发音训练
- 语言理解训练
- 交流技巧训练

### 晚期康复（发病后6个月-1年）

#### 目标
- 维持功能
- 防止退化
- 回归社会

#### 内容

- 继续功能训练
- 社会适应训练
- 职业康复（如需要）

### 康复技巧

#### 1. 肢体功能训练

**偏瘫肢体训练：**
- Bobath技术
- PNF技术
- 强制性使用运动疗法

**建议：**
- 在专业康复师指导下进行
- 循序渐进
- 坚持训练

#### 2. 言语功能训练

**失语症患者：**
- 命名训练
- 理解训练
- 表达训练

**构音障碍患者：**
- 发音训练
- 语速控制
- 语调训练

#### 3. 吞咽功能训练

**方法：**
- 冰刺激
- 舌肌训练
- 吞咽训练

**注意事项：**
- 进食时坐起，头前屈
- 小口进食
- 避免干硬食物

#### 4. 认知功能训练

**记忆训练：**
- 记忆术

- 提示策略

- 持续注意训练
- 选择性注意训练
- 分散注意训练

## 常见误区

### ❌ 误区1：脑梗患者只能卧床休息

**正确认识**：早期康复对功能恢复至关重要。生命体征稳定后即可开始康复训练，循序渐进。

### ❌ 误区2：血压越低越好

**正确认识**：脑梗急性期血压不宜过低（<140/90），以免影响脑灌注。二级预防阶段控制目标血压（<140/90）。

### ❌ 误区3：脑梗后就不能运动了

**正确认识**：在康复师指导下规律运动，有助于功能恢复和预防复发。

### ❌ 误区4：中药能根治脑梗

**正确认识**：中药可以作为辅助治疗，促进康复，但不能替代规范的西医治疗和康复训练。

### ❌ 误区5：脑梗后一定会致残

**正确认识**：及时治疗（尤其是溶栓）和规范康复，可以显著降低致残率。许多患者可以恢复正常生活。

### ❌ 误区6：只有老年人才会得脑梗

**正确认识**：虽然脑梗多发于中老年人，但近年有年轻化趋势。高血压、糖尿病、心脏病等危险因素控制不好的年轻人也可能患病。

### ❌ 误区7：脑梗后不能吃肉类

**正确认识**：应选择优质蛋白（鱼肉、去皮禽肉、豆制品），避免高脂肉类（肥肉、动物内脏）。

## 快速参考

### 脑梗救治流程

1. **识别症状**（FAST原则）
2. **立即拨打120**，不要等待
3. **等待救护车期间：**
   - 平卧，头部抬高30度
   - 解开衣领
   - 保持呼吸道通畅
   - 不要喂水、喂药（避免误吸）
   - 记录发病时间（决定能否溶栓）
4. **急诊就医**：争取黄金治疗时间窗（4.5小时内溶栓）

### 脑梗危险因素自查

请检查是否有以下危险因素：

- [ ] 年龄 > 55岁
- [ ] 男性
- [ ] 家族中有脑梗患者
- [ ] 患有高血压
- [ ] 患有糖尿病
- [ ] 患有心脏病（特别是房颤）
- [ ] 血脂异常
- [ ] 吸烟
- [ ] 肥胖（BMI ≥ 28）
- [ ] 缺乏运动
- [ ] 饮食不健康
- [ ] 过量饮酒

**如果有3个以上危险因素，属于高危人群，应：**
1. 定期体检
2. 积极控制危险因素
3. 咨询医生，必要时预防用药
