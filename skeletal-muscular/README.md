## 🔥 紧急情况

如果出现以下症状，请立即就医：

### 急性损伤
- 🦴 明显骨折（肢体变形、无法活动）
- 🦴 关节脱位（关节畸形、剧烈疼痛）
- 🦴 严重肌肉撕裂

### 神经压迫症状
- 🔴 下肢无力、麻木
- 🔴 大小便失禁（马尾综合征，紧急！）
- 🔴 进行性加重的神经症状

### 感染症状
- 🦠 关节红肿、发热、剧烈疼痛
- 🦠 全身发热、寒战

## 🛡️ 预防建议

### 关节保护
- 控制体重，减轻关节负担
- 避免过度使用关节
- 适度运动（游泳、骑自行车）
- 注意保暖，避免关节受凉

### 骨骼健康
- 补充钙质和维生素D
- 适量晒太阳
- 规律运动，增强骨密度
- 预防跌倒（老年人）

### 肌肉健康
- 运动前充分热身
- 运动后适当拉伸
- 避免突然的剧烈运动
- 保持良好姿势
