## 🔥 紧急情况

如果出现以下症状，请立即就医或拨打急救电话（120）：

### 中风症状（FAST原则）
- 🧠 Face：面部歪斜，一侧下垂
- 🧠 Arm：一侧手臂无力或麻木
- 🧠 Speech：言语不清或理解困难
- 🧠 Time：立即就医，时间就是大脑

### 严重头痛
- 🤯 突然出现的剧烈头痛（雷击样头痛）
- 🤯 伴有呕吐、意识模糊
- 🤯 以前从未有过的头痛性质

### 癫痫发作
- ⚡ 持续超过5分钟的癫痫发作
- ⚡ 连续多次发作，意识不清
- ⚡ 发作期间受伤

## 🛡️ 预防建议

### 脑血管疾病预防
- 控制高血压、糖尿病、高血脂
- 戒烟限酒
- 健康饮食
- 规律运动
- 保持良好心态

### 退行性疾病预防
- 保持大脑活跃（阅读、学习、社交）
- 规律运动
- 健康饮食（地中海饮食）
- 保证充足睡眠
- 控制心血管危险因素

### 头痛预防
- 规律作息
- 避免诱因（如特定食物、压力）
- 适度运动
- 保持良好姿势
- 管理压力
