## 🔥 紧急情况

如果出现以下症状，请立即就医：

### 糖尿病急性并发症
- 🍬 高血糖危象（DKA）
  - 口渴、多饮、多尿加重
  - 恶心、呕吐、腹痛
  - 呼吸有烂苹果味
  - 意识模糊、昏迷

- 🍬 低血糖
  - 心慌、手抖、出汗
  - 饥饿感、焦虑
  - 意识模糊、抽搐、昏迷

### 甲状腺危象
- 🦋 高热（>39°C）
- 🦋 极度烦躁、谵妄
- 🦋 心动过速（>140次/分钟）
- 🦋 大汗淋漓

## 🛡️ 预防建议

### 糖尿病预防
- 控制体重（BMI 18.5-23.9）
- 健康饮食（低糖、低脂、高纤维）
- 规律运动（每周至少150分钟）
- 定期体检（监测血糖）

### 甲状腺疾病预防
- 适量补碘（根据地区情况）
- 避免过度压力
- 规律作息
- 定期体检（甲状腺功能检查）

### 肥胖预防
- 合理膳食，控制热量摄入
- 增加体力活动
- 保证充足睡眠
- 保持良好心态
